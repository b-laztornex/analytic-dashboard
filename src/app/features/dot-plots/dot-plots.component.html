LV Volume vs Ejection Fraction ℹ Scatter of LV end-diastolic volume versus ejection fraction. Dashed lines and shaded area indicate reference EF thresholds.
Ejection Fraction by Group (Strip Plot) ℹ Jittered dots show EF distribution per group with an average line. Useful to compare central tendency and spread.
HRV Distribution (Violin-like + Dots) ℹ Approximate violin density with overlaid dots for HRV values per group. Includes a dashed median reference.
Arrhythmia Events Timeline ℹ Dots along a time axis for detected events. Highlights an initial baseline window and key time points.
Bubble Chart (QRS vs PR, size=risk) ℹ QRS vs PR intervals with bubble size mapping to risk or intensity. Reference lines mark typical thresholds.
Polar Dot Plot (Activation Phase) ℹ Dots in polar coordinates (theta vs radius) to visualize activation phase distribution.
Correlation Matrix ℹ Heatmap of pairwise correlations with dot overlay sized by magnitude. Helps identify strongest relationships.
Swarm Plot (jittered) ℹ Jittered dots show value distribution within each group with a dashed median line.
